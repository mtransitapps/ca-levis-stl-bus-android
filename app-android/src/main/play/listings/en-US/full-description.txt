This application adds Lévis STL buses information to MonTransit.

This app contains the buses schedule and the latest news from www.stlevis.ca and @STLevis on Twitter.

STL serves Lévis in Quebec, Canada.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link https://bit.ly/MonTransitPlay

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by the Société de transport de Lévis.
https://www.stlevis.ca/stlevis/donnees-ouvertes

This application is free and open-source:
https://github.com/mtransitapps/ca-levis-stl-bus-android

This app is not related with Société de transport de Lévis.

Permissions:
- Other: required to read news from www.stlevis.ca and Twitter.
